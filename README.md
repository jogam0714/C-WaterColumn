# Readme

**수행평가 관리 시스템**

*프로젝트 소개*

이 프로젝트는 C언어로 만든 콘솔 기반 수행평가 일정 관리 프로그램입니다.

학생들이 수행평가 일정을 직접 추가하고, 월별 달력으로 확인하며, 수행평가까지 남은 D-day를 계산하고, 필요없는 일정을 삭제할 수 있도록 설계되었습니다.

*주요 기능*

1. 일정 추가

사용자는 수행평가의 과목명, 수행평가 이름, 월, 일, 교시를 입력하여 일정을 등록할 수 있습니다.

입력 가능한 정보는 다음과 같습니다.

- 과목
- 수행평가 이름
- 월
- 일
- 교시

프로그램은 잘못된 입력을 방지하기 위해 월은 1월부터 12월까지만 입력할 수 있도록 제한하고, 일은 해당 월에 존재하는 날짜만 입력할 수 있도록 검사합니다. 또한 교시는 1교시부터 7교시까지만 입력할 수 있습니다.

1. 전체 일정 보기

사용자가 확인하고 싶은 월을 입력하면 해당 월의 달력이 출력됩니다.

수행평가가 있는 날짜는 별표 표시로 강조되어 한눈에 일정을 확인할 수 있습니다.

예를 들어 수행평가가 있는 날짜는 다음과 같이 표시됩니다.

- 15*

달력 아래에는 해당 월에 등록된 수행평가 일정 목록이 함께 출력됩니다.

1. D-day 보기

사용자가 오늘 날짜를 입력하면, 등록된 수행평가까지 며칠이 남았는지 계산하여 보여줍니다.

예시 출력:

영어발표D-5

수학시험D-12

이를 통해 학생은 수행평가 준비 기간을 쉽게 확인할 수 있습니다.

1. 일정 삭제

등록된 수행평가 목록 중 삭제하고 싶은 일정을 선택하여 삭제할 수 있습니다.

삭제 후에는 뒤에 있는 일정들이 앞으로 이동되어 일정 목록이 정리됩니다.

1. 종료

메뉴에서 0을 입력하면 프로그램이 종료됩니다.

*사용 방법*

프로그램을 실행하면 다음과 같은 결과가 나옵니다.

 2.전체 일정 보기 

 3.D-day보기

*실행예시*

과목 이름 : 영어

수행이름 입력: 발표

일 입력 : 20

교시 입력 : 5

위와 같이 입력하면 6월 20일 5교시에 영어 발표 수행평가 일정이 추가됩니다.

또한

*월별 일정 확인 예시*

몇 월을 입력하시겠습니까? 6
그러면 6월 달력이 출력되고, 수행평가가 있는 날짜가 별표로 표시됩니다.

## *프로그램 구조*

이 프로그램은 다음과 같은 함수들로 구성되어 있습니다.

### `add_schedule()`

수행평가 일정을 추가하는 함수입니다.

과목, 수행평가 이름, 월, 일, 교시를 입력받고 입력값이 올바른지 검사한 후 일정을 저장합니다.

### `schedule_visual()`

사용자가 입력한 월의 달력을 출력하는 함수입니다.

등록된 수행평가가 있는 날짜는 별표로 표시하고, 달력 아래에 해당 월의 수행평가 목록을 출력합니다.

### `Dday()`

오늘 날짜를 기준으로 각 수행평가까지 남은 날짜를 계산하는 함수입니다.

### `schedule_del()`

등록된 수행평가 목록을 보여주고, 사용자가 선택한 일정을 삭제하는 함수입니다.

### `main()`

프로그램의 메인 메뉴를 출력하고, 사용자가 선택한 기능에 따라 각 함수를 실행합니다.

## 제한 사항

현재 프로그램은 최대 50개의 수행평가 일정만 저장할 수 있습니다.

또한 프로그램을 종료하면 입력했던 일정이 저장되지 않기 때문에, 다시 실행하면 일정이 초기화됩니다.

또한 C언어의 배열, 함수, 조건문, 반복문, 문자열 처리 등을 실제 프로그램에 적용해 보는 것을 목적으로 합니다.
